개인정보처리방침

최종 수정일: 2026년 5월 22일

개요

본 개인정보처리방침은 '팔로워 내놔'(이하 "서비스") 이용 시 사용자의 정보를 어떻게 수집하고 이용하는지 안내합니다. 본 서비스는 "있는 그대로(AS IS)" 제공되는 무료 서비스입니다.

1. 정보 수집 및 이용

서비스 이용 시 다음과 같은 정보가 자동으로 수집될 수 있습니다.

  • 접속 기기의 IP 주소
  • 방문한 페이지, 방문 일시, 해당 페이지 머문 시간
  • 서비스 이용 시간
  • 사용 중인 운영체제 및 브라우저 정보

본 서비스는 기기의 정확한 위치 정보를 수집하지 않습니다. 또한, 사용자의 데이터를 처리하거나 기능을 제공하기 위해 인공지능(AI) 기술을 사용하지 않습니다.

2. 쿠키(Cookie) 사용 안내

저희는 사용자에게 최적화된 경험을 제공하기 위해 '쿠키'를 사용합니다. 쿠키는 웹사이트가 사용자의 브라우저에 저장하는 작은 데이터 조각입니다.

  • 사용 목적: 에디터의 환경 설정(예: 메뉴 순서 등)을 기억하여 다음 방문 시 편리하게 이용할 수 있도록 돕습니다.
  • 거부 방법: 사용자는 브라우저 설정을 통해 쿠키 저장을 거부하거나 삭제할 수 있습니다.

3. 제3자 서비스 이용

서비스 개선 및 분석을 위해 익명화된 통계 데이터가 외부 서비스에 전송될 수 있습니다. 본 서비스는 다음과 같은 제3자 서비스를 이용하며, 각 서비스의 개인정보처리방침을 따릅니다.

  • Google Analytics / Google AdSense

4. 데이터 보관 및 삭제

사용자가 직접 제공한 데이터는 서비스 이용 기간 동안 보관되며, 이후 합리적인 기간 내에 파기됩니다. 데이터 삭제를 원하실 경우 아래 이메일로 연락해 주시기 바랍니다.

5. 보안

저희는 사용자의 정보 기밀성을 보호하기 위해 물리적, 전자적, 절차적 보호 조치를 제공하며 정보 보안을 위해 최선을 다하고 있습니다.

6. 아동 보호

본 서비스는 13세 미만의 아동으로부터 고의로 정보를 수집하거나 마케팅을 하지 않습니다. 만약 13세 미만 아동의 개인정보가 수집된 사실을 인지할 경우 즉시 서버에서 삭제합니다.
